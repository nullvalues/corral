## Strategic intent

Redesign the asp reference SPA from a utilitarian table UI into a polished, mobile-first, gamified experience portfolio for applicants, plus a gamified desktop workspace for mentors. Adopt the Oklahoma State University visual system (OSU orange + warm near-black; Bricolage Grotesque + Hanken Grotesk). Add readiness scoring, configurable per-category goals, stored milestones, derived mentor impact stats, and a grant-scoped (ABAC) talent pool with reviewer-owned shortlisting. Decisions register: docs/agreements/momentum-redesign.md. NOTE: the typographic 'O' mark is a placeholder; block external/real use on the official OSU lockup per OSU brand guidelines.
