## Strategic intent

Harden the reference implementation to production-quality: close all CER forcing functions, achieve comprehensive automated E2E coverage across every role and access-control path, and deliver a human-executable UAT harness that any tester can use without knowing application internals. The terminal milestone is a signed-off UAT against the full stack — proving the reference is complete, not just compilable.

| Rail | Primary domain |
